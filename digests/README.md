# 科技简报归档

此目录包含每日生成的科技简报文件。

## 文件说明

| 文件 | 说明 |
|------|------|
| `YYYY-MM-DD.md` | 指定日期的简报 |
| `latest.md` | 最新一期简报 |
| `index.html` | HTML 格式索引页 |

## 内容来源

- **V2EX** - 技术社区热门话题
- **Hacker News** - 全球科技新闻
- **36氪** - 科技商业报道
- **少数派** - 效率工具与数码生活
- **虎嗅** - 商业科技分析
- **InfoQ中国** - 技术深度内容
- **开源中国** - 开源动态
- **Solidot** - 科技新闻

## 自动更新

简报每天北京时间 6:00 自动生成，由 GitHub Actions 驱动。
